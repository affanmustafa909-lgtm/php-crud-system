# Simple PHP CRUD System

## Overview

This is a simple CRUD (Create, Read, Update, Delete) system built using **PHP and MySQL**.
The purpose of this project is to understand the basic concept of CRUD operations and how PHP interacts with a MySQL database.

This project was created for **learning and practicing basic backend development**.

---

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* Bootstrap (optional if you used it)
* XAMPP / Localhost

---

## Features

* Add new product / data
* View all records
* Update existing records
* Delete records
* Simple and beginner-friendly structure

## How to Run the Project

1. Install **XAMPP** or any local server.
2. Copy the project folder into the **htdocs** directory.

```
xampp/htdocs/crud-system
```

3. Open **phpMyAdmin**.

4. Create a new database.

5. Import the SQL file from the `database` folder.

6. Run the project in the browser:

```
http://localhost/crud-system
```
